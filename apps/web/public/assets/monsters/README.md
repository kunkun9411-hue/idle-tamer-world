# Freigegebene Monster-Sprites

Nur vom Spiel geladene Runtime-Assets liegen hier.

- HD-/Cel-Shading-Stil, keine Pixelkunst
- PNG mit Alpha
- exakt 200×200 Pixel
- Namensschema: `<monster-id>_<animation>_<richtung>.png`

Beispiele: `pyrook_idle_right.png`, `mossbit_idle_left.png`.

Chroma-Quellen, HD-Master, PixelLab-Versuche und verworfene Stilrichtungen gehören unter `art-source/`.
