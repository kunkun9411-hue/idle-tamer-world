# UI-Arbeitsbereich

Dieser Ordner ist der schnelle Einstieg für die sichtbare Oberfläche von Idle Tamer World.

- `SCENE_INVENTORY.md` – vollständige Bestandsaufnahme aller aktuellen Spielerflächen, Zustände, Probleme und Roadmap-B-Prioritäten
- `HANDOFF.md` – Schnellstart, Dev-URLs, Audit- und Screenshotbefehle für die tägliche UI-Arbeit
- `AVATAR_FRAME_CONTRACT.md` – verbindlicher Daten-, Bild- und Zustandsvertrag für Profilbilder und Rahmen
- `../UI_SYSTEM.md` – bestehende Silver-Ether-Tokens, Komponenten und Layoutregeln
- `../ROADMAP_B_DESIGN_UI.md` – Reihenfolge der acht Design- und UI-Blöcke
- `../PLAYER_ENTRY_FLOW.md` – verbindlicher Weg von Login und Offline-Bericht in die Kampfszene

Neue UI-Entscheidungen werden zuerst einem Eintrag im Szeneninventar und anschließend einem Block der Roadmap B zugeordnet. So bleiben Problem, Lösung und Abnahme nachvollziehbar getrennt.
